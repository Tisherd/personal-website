# andrey_ivanov_portfolio
This is a resume project, optional for test tasks

# Used technologies:
- PHP 8.4
- Laravel 11 (Inertia)
- Vue 3
- PostgreSQL 17
- Redis 7

# Used dev tools:
- Laravel Telescope

# Installation
## Docker compose
- docker compose up --build
  
Inside docker container_php (docker exec -it andrey_ivanov_portfolio_php bash):
- php artisan migrate
- php artisan storage:link
